# Running
In the Terminal Go the path where DESIGNPRINCIPLES folder exists in your system.
# Windows
        Path:   C:\Users\Username\Desktop\DESIGNPRINCIPLES
        Compiling:  javac -cp . src\*.java test\*.java
        Executing:  java -cp . test.Client
# Mac/Linux
        Path:   C:/Users/Username/Desktop/DESIGNPRINCIPLES
        Compiling:  javac -cp . src/*.java test/*.java
        Executing:  java -cp . test.Client
